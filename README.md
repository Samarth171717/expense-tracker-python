# Expense Tracker (Python OOP)

A command-line Expense Tracker built using Python and Object-Oriented Programming.

## Features

* Add expenses
* View all expenses
* Calculate total expenses
* Delete expenses
* Search expenses by category
* Store expenses in a file
* Load expenses automatically on startup
* Date tracking
* Error handling

## Technologies Used

* Python 3
* Object-Oriented Programming (OOP)
* File Handling
* Exception Handling

## How to Run

```bash
python main.py
```

## Menu

1. Add Expense
2. View Expenses
3. Total Expenses
4. Delete Expense
5. Search Expense
6. Exit
